# K&A Coffee Truck Website
abdallah ahmed
12/7/2025

This is a multi-page website for **K&A Coffee Truck**, built with HTML and CSS.

## Pages

- index.html (Home)
- menu.html (Menu)
- locations.html (Locations & Schedule)
- about.html (About)
- gallery.html (Gallery)
- contact.html (Contact form)

The site uses:

- External CSS (main.css)
- Responsive design (media queries)
- Accessibility best practices (skip link, alt text, aria-current)
- SEO meta tags
- HTML table, list, and form with basic validation
